## ✨ Key Features Added

### Animations
- `animate-fade-in-up` - Fade with 30px slide up
- `animate-hero-title` - Hero title (0.8s, 0.1s delay)
- `animate-hero-subtitle` - Subtitle (0.8s, 0.2s delay)
- `animate-hero-buttons` - Buttons (0.8s, 0.3s delay)
- `btn-lift` - Button hover lift effect (-4px)
- `glow-on-hover` - Glowing effect on buttons

### Shadows
- `shadow-soft` - Subtle: 0 10px 30px rgba(0,0,0,0.08)
- `shadow-soft-lg` - Medium: 0 15px 40px rgba(0,0,0,0.12)
- `shadow-soft-xl` - Large: 0 20px 50px rgba(0,0,0,0.15)

### Effects
- Glassmorphism with 12px backdrop blur
- Gradient text on "Adventure"
- Shimmer overlay on buttons
- Scale animations on social icons
- Animated nav underlines

### Colors
- Primary: Blue #4F9EF8, #3B82F6
- Secondary: Indigo #5B5FFF, #7C3AED
- Accent: Purple #8B5CF6
- Gradients: Multi-stop combinations
